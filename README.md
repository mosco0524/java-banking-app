# 💳 Java Banking App

A console-based banking application built in Java, designed to simulate basic banking operations using object-oriented programming principles.

---

## 🚀 Features

- Account creation
- Deposit and withdrawal system
- Balance tracking
- Transaction history
- Input validation and error handling
- Transaction limits for security

---

## 🛠️ Technologies Used

- Java
- OOP (Object-Oriented Programming)
- ArrayList for transaction storage

---
